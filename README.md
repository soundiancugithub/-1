אתיקה בבינה מלאכותית – קורס SCORM (1.2)

פרויקט זה הוא קורס מקוון אינטראקטיבי בנושא אתיקה בבינה מלאכותית, שפותח כיחידת לימוד תואמת SCORM 1.2.
הקורס מיועד להרצה בתוך מערכת LMS, וכולל תוכן לימודי, חיפוש יחידות, וטופס שביעות רצון הנשמר ל־LMS.
